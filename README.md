### ASP.NET Core で WebP 対応に合わせて img タグの拡張子を置き換えるタグヘルパーを実装する方法

WebP 形式に対応したブラウザーで、同名の WebP ファイルがサーバー上に存在する場合に、拡張子を置き換えて HTML を出力させ、 WebP ファイルが表示できない場合は src で指定されたファイルをそのまま出力させるタグヘルパーを作成します。
